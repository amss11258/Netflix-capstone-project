# Netflix-capstone-project
Netflix Content Analysis – Data Analytics Capstone Project
Tools: Python, Pandas, NumPy, Matplotlib, Seaborn, Plotly
Performed exploratory data analysis on a Netflix dataset with 8,800+ titles, analyzing content distribution, genre trends, country-wise production, and release patterns. Created visualizations to identify popular genres, top producing countries, and content growth trends to support data-driven content strategy insights.
